Instructions:

This demo is an embedded Desmos project (but without the ability to edit functions). There are multiple "sliders" that contain blue buttons. Use those to change the significance of each type of function. 2 of the blue points that are on the graph control the endpoints that determine the average rate of change, and one of the points control the tangent. Try to make the tangent line green by matching the slopes as close as possible.
