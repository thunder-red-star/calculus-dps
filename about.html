Why did I make this site?

I am making this site as the final project for Calculus. When I saw the assignment, I immediately thought "Hey, maybe I should make a video!", but then I realized that, well, I don't have a tripod or a microphone. So I did the next best thing (and I did this in preparation for Comp Sci next year), I made a website. No, not baby websites where you type stuff in and they make it for you. Real sites, because they have vast potential for what they could be. As for the demos, I used these 7 examples specifically because they gave me trouble the first time I studied Calculus. These demos, therefore, are of sentimental value to me, and I hope you will find them interesting.



How to use this site

Each site follows the same basic design: An instructions section, a demo, and a reflection/quiz section (WIP). The instructions section provides important tips into the design of the demo and explains how to use it. The demos so far are made entirely with Desmos, although this can change in the future. If the demos don't load, or if you want to see the source code, there is a button under the demo which will open the demo in a new tab. Possibly in the future (1-2 weeks), I might compile questions for each demo and create a system for grading them. These aren't ready yet but info on them can be found on the blog (also in the future). Now that you have basic info about each demo, access the directory or press one of the buttons on the navigation bar to start the demos!
