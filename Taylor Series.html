Taylor Series

 Instructions:

 This demo is an embedded Desmos project (but without the ability to edit functions).  There is a "slider" that contains a blue button. Use that to change the order of the Taylor  Polynomial. The red and green points are also draggable and you can use that to change  the point at which the function is being evaluated.
